# SAM2 Annotation Tool 产品介绍文档

## 产品概述

### 产品名称
**SAM2 Annotation Tool** - 智能图像分割标注工具

### 产品定位
基于Meta SAM2（Segment Anything Model 2）模型的智能图像标注工具，为计算机视觉数据集构建提供高效、精准的标注解决方案。

### 目标用户
- **机器学习工程师**：构建训练数据集
- **数据标注团队**：大规模图像标注任务
- **研究人员**：学术论文数据集准备
- **AI项目团队**：目标检测/语义分割项目

### 版本信息
- **当前版本**：1.0.0
- **发布日期**：2026年4月
- **技术栈**：Python 3.14 + PyQt6 + SAM2

## 核心功能

### 1. 智能分割标注

| 功能 | 描述 | 操作方式 |
|------|------|---------|
| **点点击分割** | 单点触发SAM2自动分割 | 左键点击目标区域 |
| **框选分割** | 绘制矩形框触发分割 | 拖拽绘制边界框 |
| **多边形绘制** | 手动精确标注复杂边界 | 连续点击+右键闭合 |

### 2. 标注管理

| 功能 | 描述 |
|------|------|
| **标注列表** | 实时显示所有标注对象，支持选择、删除 |
| **文本标注** | 为每个标注添加自定义描述文本 |
| **类别管理** | 12种预设类别，支持层级分类（动物/车辆/建筑等） |
| **可见性控制** | 单个标注显示/隐藏切换 |

### 3. 边界微调

| 功能 | 描述 |
|------|------|
| **顶点拖拽** | 选中标注后拖拽顶点调整边界 |
| **添加顶点** | 右键菜单在边上插入新顶点 |
| **删除顶点** | 简化多边形轮廓 |

### 4. 数据导出

| 格式 | 说明 | 应用场景 |
|------|------|---------|
| **COCO JSON** | 标准数据集格式 | 通用目标检测/分割训练 |
| **VOC XML** | Pascal VOC格式 | 传统CV任务兼容 |

## 产品特性

### 智能化特性

#### SAM2模型驱动
- **零样本分割**：无需预训练，点击即可分割任意物体
- **高置信度**：平均分割置信度 > 94%
- **自动Mask生成**：输出像素级精确分割掩码

#### 高密度多边形
| 参数 | 效果 | 顶点数量 |
|------|------|---------|
| 低密度(0.01) | 简化轮廓 | ~13顶点 |
| 中密度(0.005) | **推荐配置** | ~25-50顶点 |
| 高密度(0.002) | 精细轮廓 | ~50-100顶点 |

### 质量评估体系

#### 质量指标字段
| 指标 | 范围 | 说明 |
|------|------|------|
| quality_score | 0-1 | 综合质量评分 |
| boundary_smoothness | 0-1 | 边界平滑度 |
| vertex_count | 整数 | 多边形顶点数量 |
| vertex_density | 0-20 | 顶点密度（每100px边长） |
| confidence | 0-1 | SAM2模型置信度 |

#### 评分算法
```
quality_score = 0.5 × confidence + 0.3 × density_score + 0.2 × smoothness
```

### 用户体验特性

#### 界面设计
- **深色主题**：护眼设计，适合长时间标注
- **三栏布局**：图片列表 / 画布 / 标注面板
- **响应式缩放**：自适应窗口尺寸

#### 操作便捷性
- **快捷键支持**：Ctrl+O打开、Ctrl+S保存、Delete删除
- **历史记录**：撤销/重做功能
- **状态提示**：实时显示操作状态和图片信息

## 配置选项

### config.yaml 配置项

```yaml
# 模型配置
model:
  path: "D:/模型库/SAM2/sam2_hiera_tiny.pt"
  device: "cuda"  # cuda/cpu

# 多边形配置
polygon:
  epsilon_factor: 0.005  # 顶点密度参数
  max_vertices: 100      # 最大顶点数限制

# 质量评估配置
quality:
  enable_metrics: true   # 启用质量评估
  min_vertex_density: 5  # 最小顶点密度要求

# 功能开关
features:
  high_density_polygon: true
  quality_metrics: true
  class_hierarchy: false

# 类别配置
classes:
  - name: "cat"
    id: 5
    color: "#ff9900"
    supercategory: "animal"
  # ... 共12种预设类别
```

## 使用流程

### 标准标注流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  打开图片   │ -> │  选择工具   │ -> │  创建标注   │
│ (文件夹/单张)│    │(点/框/多边形)│    │(SAM2分割)   │
└─────────────┘    └─────────────┘    └─────────────┘
                                            │
                                            v
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  导出数据   │ <- │  确认标注   │ <- │  微调边界   │
│(COCO/VOC)   │    │ (添加文本)  │    │(拖拽顶点)   │
└─────────────┘    └─────────────┘    └─────────────┘
```

### 典型操作示例

**场景：标注一张包含猫的图片**

1. **打开图片**：点击"打开文件夹"选择图片目录
2. **SAM2分割**：在猫身上点击，自动生成分割边界
3. **选择标注**：在右侧列表点击选中该标注
4. **添加文本**：输入"这是一只猫"作为描述
5. **调整边界**：如需要，点击"微调边界"拖拽顶点
6. **确认标注**：点击"确认此标注"按钮
7. **导出数据**：点击"保存COCO格式"导出JSON文件
